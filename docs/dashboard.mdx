## Descripción

Panel principal con métricas en tiempo real del restaurante. Permite seleccionar el período de análisis y actualizar manualmente los datos.

## Controles

- Selector de período: **Hoy / Esta Semana / Este Mes**
- Botón de actualización manual

## Tabla de Últimos Pedidos

Conectada directamente a Supabase, muestra los últimos 5 pedidos con columnas: ID, Cliente, Detalle, Total, Método de Pago, Servicio, Dirección y Estado.
